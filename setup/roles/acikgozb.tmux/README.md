# acikgozb.tmux

This role is responsible from instaliing tmux on a given host.

## Requirements

This role supports the distributions below:

- Arch
- Fedora
- Debian
- Darwin

Versions of the distributions don't matter.

## Role Variables

None.

## Dependencies

None.

## Example Playbook

Here's the basic usage of this role:

```yml
- hosts: servers
  roles:
    - acikgozb.tmux
```
