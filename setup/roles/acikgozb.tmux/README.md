# acikgozb.tmux

This role is responsible from instaliing tmux on the host instance.

## Requirements

Currently, only Debian and Darwin (MacOS) operating systems are supported.

## Role Variables

None.

## Dependencies

None.

## Example Playbook

Here's the basic usage of this role:

```yml
- hosts: servers
  roles:
    - acikgozb.tmux
```
